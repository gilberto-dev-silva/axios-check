## Instalação

Clone o repositório

```bash
  git clone git@github.com:gilberto-dev-silva/axios-check.git
  cd axios-check
```

Em seguida roda os seguintes comandos

```bash
  npm install
  npm run dev
```
> Após rodar os comandos acima, a url do projeto sera exibida no console.
